An Open-Source web applicaiton for finding sponsors for your organisation/event !

Uses Hugging Face to find relevant sponsors for your organisation/event.

Before using, please add your hugging face key in the .env file.

--> How to use

1.Open the Project in VS Code or any other code editor
2. Open Terminal
3. Type - "cd vexreach"
4. Then type, npm run dev
5. Local host will open type it in your web-browser to open the web-app.
